## 1. 해결 과제 및 연구 혁신성

### 1.1 이산형 액션 헤드의 경로 불연속성 및 주행 불안정성 해결
기존의 이산 분류(Discrete Classification) 방식이 초래하는 주행 불안정성을 극복하기 위해, 매니퓰레이터 분야에서 검증된 **Flow Matching 기반 연속 제어 기법**을 모바일 도출로 이식(Transfer)함. 이를 통해 장애물 회피 시 급격한 꺾임 없는 부드러운 주행 궤적을 연속적으로 생성하는 지능을 구축함.

### 1.2 3-DOF 옴니휠(Omni-wheels) 구동 시스템 최적화
전방향 즉각 이동이 가능한 **3-옴니휠 홀로노믹(Holonomic)** 특성에 완벽히 정합된 액션 정책을 설계함. π0(Pi-zero) 모델의 정밀한 연속 제어 루프를 하드웨어 기구학적 특성과 결합하여, 실시간 고빈도(50Hz+) 모바일 내비게이션 환경을 실증함.

## 2. 기대 효과
- **고주파 실시간 제어**: VLM의 무거운 연산량과 관계없이 Action Chunking을 통한 부드러운 로봇 구동 실현.
- **기술 전이의 선도적 사례**: 매니퓰레이션 특화 모델인 π0를 모바일 내비게이션 태스크로 확장하여 기술의 범용성 입증.
- **강건한 내비게이션**: 단순 타이밍 암기가 아닌, 실제 시각적 상황 이해에 기반한 정밀한 지능형 회피 주행 구현.
